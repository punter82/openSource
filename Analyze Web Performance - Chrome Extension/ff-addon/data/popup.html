  • Redirect Time : seconds

    Redirect time is the amount of time resource redirects in the page took to load. It is best to avoid redirects in any html page. You may have noticed already the number of redirects that has happened on this page displayed above. Redirect time is calculated based on the Time after last redirect response ends minus the Time of fetch that initiated a redirect. (redirectEnd - redirectStart).

  • Unload Time : seconds

    Unload Time is the amount of time taken for the page to unload. this will usually happen on page refresh or if you came to his page using ht back or forward arrow. Unload time is calculated based on Time after unload finished minus Time just before unload started (unloadEnd - unloadStart).

  • DNS Time : seconds

    DNS or Domain Name System time is the amount of time taken for the domain lookup to happen. DNS time is calculated based on Time after domain name lookup minus Time just before domain name lookup (domainLookupEnd - domainLookupStart).

  • Connection Time : seconds

    It is the time page took to establish a connection, including TCP handshakes/retries and negotiating a SSL. The connectStart is subtracted from connectEnd to determine the connection time (connectEnd - connectStart).

  • Response Time : seconds

    Response time is the amount of time taken for the server to respond to the requests made by this page. It is Time after the end of a response or connection minus Time just before a server request (responseEnd - requestStart)

  • DOM Load Time : seconds

    Amount of time take for the creation of the DOM elements. It includes Getting all markup, replaced element content and embeds from server, Parsing the markup , Applying the CSS cascade to the markup , Rendering the page, Running all scripts that need to run on page load. It basically is Time just before document readiness completes minus Time just before readiness set to loading (domComplete - domLoading)

  • General Load Time : seconds

    It is the time from - just before the load event is fired to - when the load event is complete (loadEventEnd - loadEventStart)
